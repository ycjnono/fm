# 文件管理+数据湖

## 基础选型
springboot、mysql、clickhouse、redis、mini-os、zb

## 功能
### 1、dashboard
#### 1.1、业务服务监控、中间件服务监控
#### 1.2、业务
### 2、文件管理
#### 2.1、文件上传、下载、目录、标签
#### 2.2、文件预览
### 3、内容抽取
#### 3.1、内容抽取、结构化
#### 3.2、NPL、分词（人物、地点、时间、其他）、摘要
#### 3.3、文件向量、内容向量、去重、相似度检索
#### 3.4、关系图谱（人物、地点、时间、其他）
### 4、系统管理
#### 4.1、账号、登录、apiToken
#### 4.2、监控、日志
